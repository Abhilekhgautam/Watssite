Compile with Wats

An educational language that compiles to BRIL. Designed for clarity, simplicity, and learning the fundamentals of compiler design.

main.wats

Core Features

Simplified Syntax

Focus on compiler logic, not complex grammar. The syntax is minimal and intuitive, perfect for beginners.

Direct to BRIL IR

Compile directly to the BRIL for Intermediate Languages, a standard in compiler education.

Interactive Learning

Use the online playground to get instant feedback and see the compiled BRIL output in real-time.
